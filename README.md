Добро пожаловать в игру "Новогоднее Казино"!
Здесь вы прочувствуете атмосферу настоящего Нового Года, со всеми его неожиданностями, сбывшимися и несбывшимися желаниями, а самое главное - душевной компанией, в которую каким-то образом проникли гуси, которые хотят забрать у вас деньги, но не будем об этом...

Геймплей:

1. Пользователь вводит количество ходов симуляции (количество ложек оливье)
2. Вводит seed (рецепт оливье)
3. По желанию редактирует игроков и гусей
4. Запускается симуляция

Детали архитектуры:

1. У игроков и гусей есть баланс, но он нужен только для захода в казино. Во время симуляции, их собственный баланс не меняется, меняются лишь коллекции класса CasinoBalance в классе Casino. После окончания симуляции все изменения баланса будут сохраняться в объекты игроков и гусей. В будущем можно будет сделать несколько казино.

Запуск:

- uv venv
- .venv/bin/activate (git bash)
- uv run python -m src.main

Тесты:

- uv run pytest
